Bienvenue

Apprenez à installer, configurer et automatiser la standardisation des numéros de modèle de vos produits.

Vue d'ensemble

Éliminez le travail manuel et répétitif lors de la création de vos codes d'identification.

Le module garantit des identifiants uniques et standardisés grâce à un système de modèles (Templates) intelligent. Avec cette solution, vous éradiquez les erreurs humaines et les doublons, établissant une structure logique et évolutive pour un contrôle total de votre inventaire.

Prérequis

Assurez-vous d'avoir les permissions pour :

  • Installateur & Gestionnaire d'extensions
  • Catalogue de produits

Caractéristiques principales

Essentiel

Remplissage auto intelligent

Le système identifie le modèle par défaut et remplit automatiquement le champ "Modèle" lors de l'ouverture d'un nouveau formulaire, économisant temps et clics.

Intégrité

Sécurité & Unicité

Garantit une **identité unique** pour chaque produit, évitant les numéros en double. Verrouillez le champ Modèle pour éliminer les erreurs humaines.

Important

Traitement rétroactif

Standardisez les articles existants de la boutique. Le module génère et applique en toute sécurité des numéros de modèle à vos produits actuels.

Modèles dynamiques

Combinez préfixes, intervalles et suffixes para créer des règles distinctes par département ou catégorie de produits.

Interface multilingue

Interface intuitive avec des traductions natives en EN, PT, FR, ES et IT.

Évolutivité totale

Gérez plusieurs règles simultanément sans perte de performance dans les grandes bases de données.

Structure du Numéro de Modèle

La génération de code est modulaire et flexible, divisée en trois composants qui garantissent une traçabilité complète et l'unicité.

PréfixeABC-XYZ-
Séquentiel0001
Suffixe-ASD-QWE

Préfixes

Macro-identifiants qui précèdent le numéro séquentiel (ex: ABC-XYZ-).

  • Modulaire: Segmenté en plusieurs blocs.
  • Évolutif: Ajoutez autant de blocs que vous le souhaitez.
  • Optionnel: À utiliser uniquement si nécessaire.
  • Connexion: Nécessite um séparateur avant le chiffre.

Plage Numérique

Le cœur séquentiel obligatoire (ex: 0001), garantissant l'unicité.

  • Padding: Remplissage de zéros à gauche.
  • Variable: Longueur de chiffres personnalisable.
  • Plages: Règles spécifiques par catégorie.

Suffixes

Attributs finaux pour détailler les versions ou le statut (ex: -ASD-QWE).

  • Modulaire: Segmenté en plusieurs blocs.
  • Évolutif: Ajoutez autant de blocs que souhaité.
  • Optionnel: À utiliser uniquement si nécessaire.
  • Connexion: Nécessite um séparateur après le chiffre.

Attention: Sensibilité aux Séparateurs

Le système traite chaque caractère littéralement, liant la plage numérique à la combinaison unique de préfixes, suffixes et séparateurs. Tout changement — comme remplacer un trait d'union (-) par une barre oblique (/) — définit une nouvelle identité, redémarrant automatiquement la séquence numérique pour cet identifiant spécifique.

ABC-XYZ-0001-ASD-QWE Modèle de référence.
ABC/XYZ-0001-ASD-QWE La barre oblique modifie le préfixe : le compteur redémarre pour ce nouveau groupe.

Conseil de Standardisation :

Pour maintenir la lisibilité sur les étiquettes et les rapports, utilisez des acronymes courts pour représenter les catégories ou les marques.

Recommandé : HW-MEM-DDR4-001 (Hardware - Memory - DDR4)
À éviter : HARDWARE-MEMORY-DDR4-001

Installation

Suivez les étapes ci-dessous pour intégrer le module en toute sécurité à votre système OpenCart.

01

Téléchargement

Obtenez le module officiel directement sur le Marketplace OpenCart :

Page Officielle
02

Transfert (Upload)

Dans l'administration de votre boutique, allez dans Extensions > Installateur. Cliquez sur Charger et sélectionnez le fichier téléchargé.

03

Activation

Localisez le module dans la liste des extensions et cliquez sur l'icône Installer pour l'activer.

Conseil technique : Après l'activation, n'oubliez pas d'aller dans Extensions > Modifications et de cliquer sur le bouton Actualiser (icône bleue) pour vider le cache du système.

Accès aux Paramètres

Après l'installation, suivez ce flux de travail para configurer votre automatisation.

01

Menu Latéral

Allez dans Extensions > Extensions.

02

Filtrer

Sélectionnez le type d'extension Modules.

03

Modifier

Cliquez sur Modifier pour ouvrir le panneau.

1. Paramètres Généraux

ParamètreFonction
Remplissage autoGénère instantanément le modèle lors de la création de produits.
Empêcher l'éditionVerrouille le champ pour empêcher toute modification manuelle.
ÉtatActive ou désactive le module.

2. Préfixe et Suffixe

Ces onglets vous permettent de composer les éléments de texte ou de date entourant le numéro séquentiel.

Paramètres de Groupe

ParamètreFonction
Nom Identification interne.
Ex: Électronique, Vêtements.
Séparateur Caractère qui relie ce groupe au numéro séquentiel.

Composition de l'Élément

ParamètreDescription
Type Définit si l'élément sera un Texte Fixe ou une Date Dynamique.
Contenu (Texte) La valeur textuelle à afficher (ex: PROD).
Format (Date) Le format de date souhaité (ex: Année (2 chiffres) + mois).
Séparateur Caractère qui relie cet élément au suivant au sein du même préfixe.

Astuce : Vous pouvez ajouter plusieurs éléments pour créer des préfixes complexes, tels que ANNEE-CATEGORIE-.

3. Intervalle Séquentiel

Paramètre Description
Nom Identification interne.
Ex: Compteur Général, Lot 2024.
Zéros non significatifs Active le remplissage (padding) pour une standardisation visuelle.
Longueur Nombre total de chiffres.
Ex: Une longueur de 4 transforme "1" en "0001".
Min / Max Définit le point de départ et la limite finale du comptage.

Note de Sécurité : Le système utilise ces limites pour garantir la non-duplication. Une fois la valeur maximale atteinte, une alerte empêchera de nouveaux enregistrements pour garantir l'intégrité des données.

4. Modèle (Template)

Paramètre Description
Nom Identification interne.
Ex: Souris, Clavier, Feuilles A4.
Préfixe Lien vers le groupe de Préfixe configuré.
Intervalle Lien vers la règle de Numérotation Séquentielle.
Suffixe Lien vers le groupe de Suffixe configuré.
Par défaut Définit ce modèle comme principal pour le remplissage auto.

Astuce de Flux : Le Modèle est l'endroit où vous "assemblez" les paramètres précédents. Assurez-vous que les groupes de Préfixe, Intervalle et Suffixe ont déjà été créés avant de finaliser cette étape.

Génération de Numéros

Suivez les étapes ci-dessous pour appliquer une numérotation automatique à vos produits.

01

Navigation

Dans le menu latéral, allez dans Catalogue et sélectionnez l'option Produits.

02

Accès

Cliquez sur Modifier sur un produit ou sur le bouton Ajouter.

03

Emplacement

Allez dans l'onglet Données et localisez le champ Modèle dans le formulaire.

04

Générer

Sélectionnez le template et cliquez sur le bouton Générer . Le champ Modèle sera rempli.

Astuce Pratique : Si vous sélectionnez un template non défini par défaut et cochez l'option "Définir par défaut", le système enregistrera automatiquement votre choix lors de la génération.

Désinstallation

Suivez les étapes ci-dessous pour une désinstallation propre et sécurisée.

01

Désactiver

Allez dans Extensions > Extensions, filtrez par Modules, localisez le module et cliquez sur Désinstaller.

02

Désinstaller

Localisez le module dans la liste des extensions installées et cliquez sur l'icône Désinstaller.

03

Supprimer

Toujours dans la liste des extensions installées, cliquez sur Supprimer.

Qu'advient-il des données ?

La désinstallation supprime les paramètres et les fichiers du module. Cependant, les numéros de modèle déjà générés pour vos produits resteront enregistrés dans la base de données pour éviter toute perte d'intégrité de vos dossiers.
