Bem-vindo

Aprenda a instalar, configurar e automatizar a padronização dos números de modelos de seus produtos.

Visão Geral

Elimine o trabalho manual e repetitivo na criação de códigos de identificação.

O módulo assegura identificadores exclusivos e padronizados através de um sistema inteligente de Templates. Com esta solução, você erradica erros humanos e duplicidades, estabelecendo uma estrutura lógica e escalável para o controle total do seu inventário.

Requisitos

Certifique-se de ter permissões em:

  • Instalador e Gerenciador de Extensões
  • Catálogo de Produtos

Recursos Principais

Essencial

Autopreenchimento Inteligente

O sistema identifica o template padrão e preenche o campo "Modelo" automaticamente ao abrir um novo formulário, economizando tempo e cliques.

Integridade

Segurança e Unicidade

Garante **identidade única** para cada produto, impedindo a geração de números duplicados. Trave o campo Modelo para edição manual, eliminando erros humanos.

Importante

Processamento Retroativo

Padronize itens antigos da loja. O módulo gera e aplica números de modelos para produtos existentes de forma segura.

Templates Dinâmicos

Combine prefixos, intervalos e sufixos para criar regras distintas por departamento ou categoria de produto.

Interface Multi Idiomas

Interface intuitiva com traduções nativas em PT, EN, FR, ES e IT.

Escalabilidade Total

Gerencie múltiplas regras simultaneamente sem perda de performance em grandes bancos de dados.

Estrutura do Número de Modelo

A formação do código é modular e flexível, dividida em três componentes que garantem rastreabilidade total e exclusividade.

PrefixoABC-XYZ-
Sequencial0001
Sufixo-ASD-QWE

Prefixos

Identificadores macro que precedem o número sequencial (ex: ABC-XYZ-).

  • Modular: Segmentado em múltiplos blocos.
  • Escalável: Adicione quantos blocos desejar.
  • Opcional: Utilize apenas quando necessário.
  • Conexão: Exige separador antes do numeral.

Intervalo Numérico

O núcleo sequencial obrigatório (ex: 0001), garantindo a exclusividade.

  • Padding: Preenchimento de zeros à esquerda.
  • Variável: Comprimento de dígitos customizável.
  • Faixas: Regras específicas por categoria.

Sufixos

Atributos finais para detalhar versões ou status (ex: -ASD-QWE).

  • Modular: Segmentado em múltiplos blocos.
  • Escalável: Adicione quantos blocos desejar.
  • Opcional: Utilize apenas quando necessário.
  • Conexão: Exige separador antes do numeral.

Atenção: Sensibilidade aos Separadores

O sistema processa cada caractere de forma literal, vinculando o intervalo numérico à combinação exclusiva de prefixos, sufixos e separadores. Qualquer alteração — como a troca de um hífen (-) por uma barra (/) — define uma nova identidade, reiniciando automaticamente a sequência numérica para esse identificador específico.

ABC-XYZ-0001-ASD-QWE Padrão de referência.
ABC/XYZ-0001-ASD-QWE A barra altera o prefixo: contagem reinicia para este novo grupo.

Dica de Padronização:

Para manter a legibilidade em etiquetas e relatórios, utilize siglas curtas para representar categorias ou marcas.

Recomendado: HW-MEM-DDR4-001 (Hardware - Memória - DDR4)
Evite: HARDWARE-MEMORIA-DDR4-001

Instalação

Siga as etapas abaixo para integrar o módulo ao seu sistema OpenCart de forma segura.

01

Download

Obtenha o módulo oficial diretamente no marketplace do OpenCart:

Página Oficial
02

Upload

No painel da loja, acesse Extensões > Instalador. Clique em Upload e selecione o arquivo baixado.

03

Ativação

Localize o módulo na lista de extensões instaladas e clique no ícone de Instalar para ativá-lo.

Dica Técnica: Após a ativação, lembre-se de acessar Extensões > Modificações e clicar no botão Atualizar (ícone azul) para limpar o cache do sistema.

Acesso às Configurações

Após a instalação, siga este fluxo para configurar sua automação.

01

Menu Lateral

Acesse Extensões > Extensões.

02

Filtro

Selecione o tipo Módulos.

03

Edição

Clique em Editar para abrir o painel.

1. Configuração Geral

ParâmetroFunção
Preenchimento AutomáticoGera o modelo instantaneamente ao criar produtos.
Impedir EdiçãoTrava o campo para evitar alterações manuais.
StatusHabilita o módulo.

2. Prefixo e Sufixo

Estas abas permitem compor os elementos de texto ou data que cercam o número.

Configurações do Grupo

ParâmetroFunção
Nome Identificação interna.
Ex: Eletrônicos, Vestuário.
Separador Grupo Caractere que une este grupo ao número sequencial.

Composição do Elemento

ParâmetroDescrição
Tipo Define se o elemento será um Texto Fixo ou uma Data Dinâmica.
Conteúdo (Texto) O valor textual que será exibido (ex: PROD).
Formato (Data) O padrão de data desejado (ex: Ano dois dígitos + mês).
Separador Interno Caractere que unirá este elemento ao próximo dentro do mesmo prefixo.

Dica: Você pode adicionar múltiplos elementos para criar prefixos complexos, como ANO-CATEGORIA-.

3. Intervalo Sequencial

Parâmetro Descrição
Nome Identificação interna.
Ex: Contagem Geral, Lote 2024.
Zeros à esquerda Habilita o preenchimento para padronização visual.
Comprimento Total de dígitos.
Ex: Comprimento 4 transforma "1" em "0001".
Mínimo / Máximo Define o ponto de partida e o limite final da contagem.

Dica de Segurança: O sistema utiliza estes limites para controlar a não duplicidade. Ao atingir o valor máximo, será emitido um alerta impedindo novos registros para garantir a integridade dos seus dados.

4. Modelo (Template)

Parâmetro Descrição
Nome Identificação interna.
Ex: Mouse, Teclado, Folhas A4.
Prefixo Vínculo com o grupo de **Prefixo** configurado.
Intervalo Vínculo com a regra de **Numeração Sequencial**.
Sufixo Vínculo com o grupo de **Sufixo** configurado.
Padrão Define o template como principal para o **autopreenchimento**.

Dica de Fluxo: O Template é onde você "costura" as configurações anteriores. Certifique-se de que os grupos de Prefixo, Intervalo e Sufixo já foram criados antes de finalizar esta etapa.

Gerar Números

Siga os passos abaixo para aplicar a numeração automática aos seus produtos.

01

Navegação

No menu lateral, vá em Catálogo e selecione a opção Produtos.

02

Acesso

Clique em Editar no produto ou no botão Adicionar.

03

Localização

Acesse a aba Dados e identifique o campo Modelo no formulário.

04

Gerar número

Selecione o template e clique no botão Gerar . O campo Modelo será preenchido.

Dica de Praticidade: Ao selecionar um template não padrão e marcar a opção "Definir como padrão", o sistema salvará sua escolha automaticamente ao gerar o número.

Desinstalação

Siga os passos abaixo para uma desinstalação limpa e segura.

01

Desativar

Vá em Extensões > Extensões, filtre por Módulos, localize o módulo e clique em Desinstalar.

02

Desinstalar

Localize o módulo na lista de extensões instaladas e clique no ícone Desinstalar.

03

Excluir

Ainda na lista de extensões instaladas e clique em Excluir.

O que acontece com os dados?

A desinstalação remove as configurações e arquivos do módulo. No entanto, os números de modelo já gerados nos seus produtos permanecerão salvos no banco de dados para evitar perda de integridade nos seus cadastros.
